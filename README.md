# Coco Accelerated
A hardware accelerated implementation of the [Coco BBOB suite](https://github.com/numbbo/coco).

## Requirements

For everything to work properly you need
- [Rust](https://www.rust-lang.org/)
- [Futhark](https://futhark-lang.org/)
- GCC or Clang
- Cuda SDK (optional)
- OpenCL SDK (optional)
